# To-Do App

A simple desktop To-Do list application built with Python and Tkinter.

## Features

- Add tasks by typing in the input field and pressing **Enter** or clicking the **Add** button
- View all tasks in a scrollable list
- Delete individual tasks with the **Delete** button next to each item
- Warns you if you try to add an empty task

## Requirements

- Python
- Tkinter (included in the standard Python distribution; on some Linux systems you may need to install it separately)

### Installing Tkinter on Linux

```bash
sudo apt-get install python3-tk
```

## Running the App

```bash
python todo.py
```

> **Note:** No external dependencies or virtual environment are required — the app uses only the Python standard library.
